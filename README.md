# GitHub Final Project

## Descripción

Este repositorio contiene el proyecto final del curso de GitHub de Coursera. Es un proyecto de demostración que incluye un calculador de interés simple implementado en Bash.

## Características

- Script de calculadora de interés simple en Bash
- Licencia Apache 2.0
- Documentación completa
- Código de conducta para contribuyentes
- Guías de contribución

## Uso

Para usar el calculador de interés simple:

```bash
bash simple-interest.sh
```

## Fórmula del Interés Simple

El interés simple se calcula usando la fórmula:

```
Interés Simple = (Principal × Tasa × Tiempo) / 100
```

Donde:
- Principal = Cantidad inicial de dinero
- Tasa = Tasa de interés anual (%)
- Tiempo = Tiempo en años
